# AutoWork 배포 가이드

## 배포 방식: GitHub Pages

### 사전 요구사항
1. GitHub 저장소: https://github.com/aebonlee/autowork
2. CNAME 파일: `public/CNAME` → `autowork.dreamitbiz.com`
3. 404.html: SPA 리다이렉트 스크립트 포함
4. Supabase Auth에 redirect URL 등록

### 빌드 & 배포 명령어
```bash
# OG 이미지 생성 (최초 1회 또는 변경 시)
npm run generate-og

# 빌드
npm run build

# GitHub Pages 배포
npx gh-pages -d dist
```

### Git 초기화 (최초 1회)
```bash
git init
git remote add origin https://github.com/aebonlee/autowork.git
git add .
git commit -m "Initial commit: AutoWork 업무자동화 학습 플랫폼"
git branch -M main
git push -u origin main
```

### DNS 설정
- dreamitbiz.com DNS에 CNAME 레코드 추가:
  - `autowork.dreamitbiz.com` → `aebonlee.github.io`

### Supabase 설정
1. Supabase SQL Editor에서 `supabase-setup.sql` 실행
2. Authentication > URL Configuration에 redirect URL 추가:
   - `https://autowork.dreamitbiz.com`
3. OAuth Providers (Google, Kakao) 설정 확인
