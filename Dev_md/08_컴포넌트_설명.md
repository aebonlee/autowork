# AutoWork 컴포넌트 상세 설명

## Context Providers

### ThemeContext
- 5색 테마 (blue/red/green/purple/orange)
- 3모드 (auto/light/dark)
- localStorage 저장, CSS data-theme 속성 제어

### LanguageContext
- ko/en 전환, 쿠키 저장
- `t(key)` 함수로 번역 텍스트 접근
- translations.js에서 번역 데이터 로드

### AuthContext
- Supabase PKCE 인증
- onAuthStateChange 리스너
- 프로필 로딩, 계정 상태 확인
- Google/Kakao/Email 로그인 지원

### ToastContext
- 4종 알림: success, error, warning, info
- 3초 자동 닫기, 수동 닫기 가능

## 레이아웃 컴포넌트

### Navbar
- 글래스모피즘 디자인, 스크롤 감지
- 로고: "Auto" + "Work" (볼드+컬러)
- 7개 메뉴, 컬러피커, 테마토글, 언어토글
- 모바일: 햄버거 메뉴 (1100px 이하)
- 유저 메뉴: 프로필/대시보드/설정/로그아웃

### Footer
- 3열: 브랜드/빠른링크/연락처+Family Site
- Family Site: DreamIT Biz, Edu-Hub, Teaching AI
- 다크 그라데이션 배경

### HeroCarousel
- 4슬라이드: particles, network, orbs, geometric
- 6초 자동재생, 수동 네비게이션
- 배지 + 타이틀 + 설명 + CTA 버튼

## 페이지 컴포넌트

### Home
- HeroCarousel + 카테고리 카드 (10개) + 통계 + CTA
- 통계: 10 카테고리 / 50+ 레슨 / 5 테마 / 2 언어

### IntroPage (ck-* 패턴)
- 사이드바: 사이트 안내 > About/사용가이드 | 학습 로드맵 | 도구 소개
- AboutSection: 4개 특징 카드 + 바로가기 링크
- GuideSection: 6단계 사용 가이드
- RoadmapSection: 입문→중급→고급 경로
- ToolsSection: Excel, Python, RPA, Zapier/Make, AI

### LessonCategories
- 10개 카테고리 카드 그리드
- 아이콘, 이름, 설명, 레슨 수, 난이도 뱃지

### LessonList
- URL의 categorySlug로 카테고리 조회
- 카테고리 헤더 + 번호 매긴 레슨 목록

### LessonDetail
- 동적 import로 콘텐츠 로딩
- ReactMarkdown + remarkGfm 렌더링
- TOC 사이드바 (h2/h3 추출)
- 이전/다음 레슨 네비게이션
- 브레드크럼: 학습 / 카테고리 / 레슨명

### Board / BoardDetail / BoardWrite
- Supabase autowork_board_posts 테이블
- 카테고리: 전체/공지/자동화 팁/질문/자유/사례 공유
- 조회수 RPC: increment_autowork_post_views
- 댓글: autowork_board_comments

### Dashboard
- 카테고리별 학습 진도 (0/N)
- 빠른 접근 그리드

### Settings
- 프로필 탭: 프로필 이미지, 닉네임, 이메일

### AdminDashboard
- 통계: 전체 사용자 / 게시글 / 댓글 수
- autowork_board_posts/comments 테이블 조회

## 가드 컴포넌트

### AuthGuard
- isLoggedIn 확인, 미인증 → /login 리다이렉트
- 로딩 중 스피너 표시

### AdminGuard
- isAdmin 확인, 비관리자 → / 리다이렉트
- ADMIN_EMAILS 목록 기반 확인
