# AutoWork 스타일 & 테마 시스템

## CSS 파일 구성 (16개)

| 파일 | 역할 |
|------|------|
| index.css | 모든 CSS 임포트 허브 |
| base.css | CSS 변수, 리셋, 버튼, 폼, 유틸리티 |
| navbar.css | 글래스모피즘 네비바 |
| hero.css | 히어로 캐러셀 + 파티클 배경 |
| footer.css | 다크 그라데이션 푸터 |
| auth.css | 로그인/회원가입 폼 |
| dashboard.css | 대시보드 레이아웃 |
| settings.css | 설정 페이지 |
| community.css | 게시판 테이블/카드 |
| admin.css | 관리자 대시보드 |
| **lessons.css** | 학습 페이지 (신규) |
| **intro-page.css** | 소개 페이지 ck-* 패턴 (신규) |
| animations.css | AOS 스크롤 애니메이션 |
| toast.css | 토스트 알림 |
| dark-mode.css | 다크모드 CSS 변수 |
| responsive.css | 반응형 미디어쿼리 |

## 5색 테마 시스템
```
blue:   --primary: #1B3A6B  (기본값)
red:    --primary: #C8102E
green:  --primary: #00855A
purple: --primary: #8B1AC8
orange: --primary: #C87200
```

## 모드 시스템
- **auto**: OS 설정 따름 (prefers-color-scheme)
- **light**: 밝은 테마
- **dark**: 어두운 테마 (dark-mode.css 변수 오버라이드)

## 반응형 브레이크포인트
| 기준 | 적용 |
|------|------|
| 1100px | 네비바 모바일 전환 |
| 1024px | 사이드바 레이아웃 → 단일 컬럼 |
| 768px | 카드 그리드 축소, 패딩 감소 |
| 480px | 폰트 축소, 단일 컬럼 |

## 글래스모피즘 네비바
```css
background: rgba(255, 255, 255, 0.85);
backdrop-filter: blur(20px);
border-bottom: 1px solid rgba(0, 0, 0, 0.06);
```

## 새로 추가된 CSS 패턴

### lessons.css
- `.lesson-categories-grid` - 카테고리 카드 그리드
- `.lesson-list-item` - 레슨 목록 아이템
- `.lesson-detail-layout` - 사이드바(TOC) + 본문 그리드
- `.markdown-body` - 마크다운 렌더링 스타일
- `.lesson-nav` - 이전/다음 네비게이션

### intro-page.css (ck-* 패턴)
- `.ck-page` / `.ck-layout` - 사이드바 페이지 레이아웃
- `.ck-sidebar` - 스티키 사이드바 (top: 100px)
- `.ck-nav-parent` / `.ck-nav-child` - 계층형 네비게이션
- `.ck-content-box` - 콘텐츠 카드
- `.intro-features-grid` - 기능 카드 그리드
- `.intro-roadmap` - 학습 로드맵 (입문→중급→고급)
