# AutoWork 라우팅 구조

## 전체 라우트 맵

| 경로 | 컴포넌트 | 인증 | 설명 |
|------|---------|------|------|
| `/` | Home | - | 메인 홈 (히어로 + 카테고리 + 통계 + CTA) |
| `/intro` | IntroPage | - | 사이트 소개 (사이드바 네비게이션) |
| `/lessons` | LessonCategories | - | 10개 카테고리 그리드 |
| `/lessons/:categorySlug` | LessonList | - | 카테고리별 레슨 목록 |
| `/lessons/:categorySlug/:lessonSlug` | LessonDetail | - | 레슨 상세 (마크다운 렌더링) |
| `/login` | Login | - | 로그인 (Google/Kakao/Email) |
| `/register` | Register | - | 회원가입 |
| `/forgot-password` | ForgotPassword | - | 비밀번호 찾기 |
| `/dashboard` | Dashboard | AuthGuard | 학습 대시보드 |
| `/settings` | Settings | AuthGuard | 사용자 설정 |
| `/community/board` | Board | - | 게시판 목록 |
| `/community/board/write` | BoardWrite | AuthGuard | 글쓰기 |
| `/community/board/:id` | BoardDetail | - | 게시글 상세 |
| `/admin/*` | AdminDashboard | AdminGuard | 관리자 대시보드 |
| `*` | NotFound | - | 404 페이지 |

## 네비게이션 메뉴 (7개)

| 순서 | 경로 | 한국어 | 영어 |
|------|------|--------|------|
| 1 | /intro | 사이트 소개 | About |
| 2 | /lessons | 학습하기 | Lessons |
| 3 | /lessons/excel | 엑셀 자동화 | Excel Auto |
| 4 | /lessons/python | Python 자동화 | Python Auto |
| 5 | /lessons/nocode | 노코드/로코드 | No-Code |
| 6 | /lessons/ai | AI 자동화 | AI Automation |
| 7 | /community/board | 커뮤니티 | Community |

## 가드 체계
- **AuthGuard**: 로그인 필요 → 미인증 시 /login 리다이렉트
- **AdminGuard**: 관리자 권한 필요 → ADMIN_EMAILS 목록 확인

## 코드 분할
모든 페이지 컴포넌트는 `React.lazy()` + `Suspense`로 동적 로딩:
```jsx
const Home = lazy(() => import('../pages/Home'));
const LessonDetail = lazy(() => import('../pages/lessons/LessonDetail'));
```

레슨 콘텐츠는 이중 동적 import:
```jsx
import(`../../content/${categorySlug}/${lessonSlug}.js`)
```
