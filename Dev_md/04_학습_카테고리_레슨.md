## 콘텐츠 파일 구조
각 레슨 콘텐츠는 `src/content/{category}/{slug}.js` 형태로 저장:
```js
const lesson = {
  titleKo: '...',
  titleEn: '...',
  contentKo: `# 마크다운 콘텐츠 (한국어)...`,
  contentEn: `# Markdown content (English)...`,
};
export default lesson;
```

동적 로딩: `import(\`../../content/${categorySlug}/${lessonSlug}.js\`)`
